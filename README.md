# simple-trello

## Project Overview
This repository contains a minimal implementation of a Trello-like application for task management.

## Features
- Create, update, and delete boards
- Add and manage cards

## Technologies Used
- HTML
- CSS
- JavaScript

## Usage
1. Clone the repository.
2. Open `index.html` in your browser.

## Future Plans
- Integrate with a backend database
